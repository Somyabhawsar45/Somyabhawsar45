## About Me

I'm a developer focused on **agentic AI systems** and full stack web development. My recent work centers on building **Netra AI**, a deployed agentic RAG chatbot using LangGraph, Groq, and FAISS — covering everything from stateful graph orchestration to retrieval pipelines for document-based Q&A. I enjoy working across the stack, from designing conversational AI architectures to shipping polished front-end interfaces.

- 🔭 Currently building and refining **Netra AI** — an agentic RAG chatbot with persistent memory

- 🌐 Comfortable across Python and JavaScript ecosystems

## Featured Projects

<details open>
<summary><b>🤖 Agentic-Chatbot (Netra AI)</b></summary>
<br/>

An agentic RAG chatbot built with **LangGraph**, **Groq**, **FAISS**, and **Streamlit**, supporting four use cases: Basic Chatbot, Chatbot With Web, AI News, and Chat with PDF.

| Aspect | Details |
|---|---|
| **Stack** | LangGraph · Groq · FAISS · Streamlit · SQLite |
| **Architecture** | Stateful graph orchestration with persistent SQLite-backed chat history injected into LLM context |
| **RAG Pipeline** | `sentence-transformers/all-MiniLM-L6-v2` embeddings, FAISS indexed per conversation ID |
| **Key Feature** | Cross-session memory — full conversation history is passed into `graph.stream()` / `graph.invoke()` |
| **Repository** | [github.com/Somyabhawsar45/Agentic-Chatbot](https://github.com/Somyabhawsar45/Agentic-Chatbot) |
